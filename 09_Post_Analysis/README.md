### Galaxy

- [Galaxy Project](https://galaxyproject.org/) - The Galaxy Project is an open-source platform for data-intensive biomedical research.
- [Galaxy Training Network](https://training.galaxyproject.org/) - A collection of training materials for using Galaxy.
- [European Galaxy Instance](https://usegalaxy.eu/) - The European Galaxy instance, which is free to use for academic research.
- [HCEMM's Galaxy Instance](https://galaxy.hcemm.eu/) - HCEMM's Galaxy instance, currently in development and available for testing.
